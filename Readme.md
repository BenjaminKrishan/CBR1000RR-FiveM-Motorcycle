# Honda CBR 1000RR (Unbranded) - FiveM Resource

![Version](https://img.shields.io/badge/Version-1.0.0-blue.svg)

A high-quality, ultra-realistic CBR 1000RR motorcycle mod for FiveM. This resource is unbranded and comes with a variety of tuning options to enhance your server's motorcycle collection.

## 🚀 Features

- **Spawn Code**: `cbr1000rr`
- **Tuning Options**:
  - Multiple Exhaust Systems
  - Custom Fairings & Guards
  - Race-tier modifications
- **High Quality Models**: Realistic textures and detailed LODs.
- **Custom Handling**: Optimized for a smooth and realistic riding experience.
- **Optimized**: Performance-friendly assets with proper texture compression.

## 🛠️ Installation

1. Download the resource.
2. Extract the `CBR1000RR` folder into your FiveM server's `resources` directory.
3. Open your `server.cfg` file.
4. Add `ensure CBR1000RR` to the list of resources to be started.
5. Restart your server or start the resource in-game.

## 📁 Project Structure

- `data/`: Contains meta files (`handling.meta`, `vehicles.meta`, `carcols.meta`, etc.).
- `stream/`: Contains all YTD and YFT asset files.
- `fxmanifest.lua`: Configuration for the resource manifest.
